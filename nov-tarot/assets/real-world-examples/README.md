# Real-world Examples 语料库

真实塔罗师的解读样本,用于校准本 skill 的解读质感。与 `assets/example-*.md`(从早期提示词搬来的样本)不同,这里的样本来自真实从业者,是"好的解读"的实证参照。

SKILL.md 不主动引用这里(避免把特定塔罗师的话术固化成模板),但需要校准解读语感、切入点、敏感处理方式时,可读这里的素材。

## 目录结构

```
real-world-examples/
├── raw/          # 原话存档:转写稿、逐字段落,保留塔罗师原话
└── distilled/    # 技法提炼:从原话里蒸馏出的解读技法(切入点、节奏、话术、收束)
```

两层都留:**raw 保留原话语感,distilled 提炼可复用技法**。原话是语料,蒸馏是洞察。

## 命名约定

`<塔罗师>-<主题或日期>-<序号>.md`,如 `maytarot-感情时间流-001.md`。

## 蒸馏什么

不是把转写稿原样塞进来(那是噪音)。从原话里提炼:
- **切入点**:她从哪里开始解读?先共情还是先点牌?
- **话术节奏**:怎么从牌面过渡到用户处境?
- **敏感处理**:遇到不好听的牌意,她怎么讲?
- **收束方式**:怎么结尾?给建议还是留余地?
- **语气特征**:仪式感怎么拿捏?神叨到什么程度?

每条蒸馏要附原话出处(指向 raw/ 里的具体段落),让技法能回溯到真实语料。
