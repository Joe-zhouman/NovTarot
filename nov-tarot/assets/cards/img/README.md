# 牌面真图目录

这个目录放韦特塔罗的牌面真图(正位图)。**当前为空**——报告生成时用占位框,见 `assets/agents/tarot-report-compiler.md` 的「卡图」一节。

## 命名约定

文件名 = 牌的 `ref` + `.jpg`,ref 见 `references/card-index.md`。

- 大阿卡纳:`the-fool.jpg`、`the-empress.jpg` ... `the-world.jpg`
- 小阿卡纳:`<花色>-<数字或宫廷>.jpg`,如 `wands-ace.jpg`、`cups-queen.jpg`、`pentacles-10.jpg`

共 78 张(22 大阿卡纳 + 56 小阿卡纳)。

## 正逆位规则

**只放正位图。** 逆位靠 CSS `transform: rotate(180deg)` 旋转得到,不需要单独的逆位素材。这是塔罗逆位的视觉语义(逆位 = 牌倒过来),也省一半素材。

## 接入方式

报告 subagent 生成 HTML 时,检查本目录下对应图是否存在:
- 存在 → 用 `<img>` 替换占位框(逆位同样旋转 180°)
- 不存在 → 继续用设计感占位框

所以往这个目录放图即可自动接入,不用改任何代码。放齐 78 张前,缺的牌会自动降级为占位框。

## 版权注意

韦特塔罗的牌面图( Pamela Colman Smith 绘制)在美国已进入公有领域(1909 年出版),但在其他司法辖区可能仍有版权。若公开发布报告含真图,确认你所在地区的版权状态,或使用自己拍摄/绘制的牌面。
