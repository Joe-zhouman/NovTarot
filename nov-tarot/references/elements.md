# 元素系统与组合解读

解读涉及多张牌的元素关系时查这份。单张牌的元素属性见各单牌 ref(`references/cards/waite/<ref>.md`),或用 `scripts/lookup-card.py` 查询。

元素是塔罗解读中最底层的"能量语言"——比守护星更直觉，比牌面符号更原始。看一组牌的元素分布，你就知道这场解读的能量基调：是火在烧还是水在浸，是风在吹还是土在压。

## 四元素详解

### 火 — 行动力、热情、意志、冲突

火是点燃。它的问题是"我在往哪走？"

- 核心领域：行动、事业、创造力、性、竞争、冒险、自我表达
- 正向表达：热情、勇气、果断、感染力、领导者气质、说做就做
- 阴影表达：冲动、暴躁、三分钟热度、攻击性、自我中心、消耗殆尽
- 在塔罗解读中：火元素牌多 → 用户处于行动模式或需要行动。火不问对不对，火只问做不做。火元素集中出现在"未来"位 → 即将进入高强度行动期；出现在"过去"位 → 被从前的冲动或热情驱动到今天。火完全缺席 → 整件事缺一股"推出去"的力量，用户可能在观望、拖延、或被外部力量牵着走。
- 逆位：火受阻——热情还在但发不出来，或方向错了在瞎烧。不是没火，是火被闷住了。
- 花色：权杖 | 星座：白羊、狮子、射手 | 守护星：火星
- 季节感：春（萌发与冲刺）

### 水 — 情感、直觉、关系、潜意识

水是浸润。它的问题不是"发生了什么"，而是"你对此感觉如何"。

- 核心领域：情感、亲密关系、直觉、梦境、创造力源头、潜意识、共情
- 正向表达：敏感、滋养、深情、直觉力、艺术感受力、与他人深度连接
- 阴影表达：情绪化、依赖、沉溺、边界模糊、被动攻击、被情绪淹没
- 在塔罗解读中：水元素牌多 → 这不是逻辑能算清的事，情感才是核心变量。问事业却水元素多 → 问题不在事，在心。水元素完全缺席 → 情感被压抑或被忽视，用户可能在用理性回避感受。尤其问感情却水缺失——这段关系可能已经干了。
- 逆位：水停滞或泛滥——停滞是情绪冻住（说不出口、哭不出来），泛滥是情绪失控（过度反应、被淹没）。区分两者看上下文：周围牌是压抑型（宝剑、逆位）→ 停滞；周围牌是爆发型（火元素、高塔）→ 泛滥。
- 花色：圣杯 | 星座：巨蟹、天蝎、双鱼 | 守护星：月亮
- 季节感：夏（丰沛与潮汐）

### 风 — 理性、思维、变革、沟通

风是穿透。它带来清晰，也带来寒冷。

- 核心领域：思维、沟通、学习、决策、真理、计划、技术、语言
- 正向表达：清晰、机智、客观、善于分析、表达力、快速适应、看到全局
- 阴影表达：过度分析、冷漠、刻薄、焦虑、空谈、精神内耗、逃避情感
- 在塔罗解读中：风元素牌多 → 用户正在用脑子处理这件事，可能过度。风元素的好处是看得清，坏处是停在"看清"——看清楚了，然后呢？风元素完全缺席 → 思维混乱、沟通不畅、或用户在做决定前没有充分想清楚。
- 逆位：风的正常状态就是动的，逆位风常常是"风向变了"——思维正在从混乱转向清晰（宝剑八逆→九正），或从清晰坠入混乱（宝剑九逆→八正）。解读逆位风时多看它在哪张牌上——宝剑牌的逆位各有特定含义，不要笼统读"思维混乱"。
- 花色：宝剑 | 星座：双子、天秤、水瓶 | 守护星：水星
- 季节感：秋（清冽与洞察）

### 土 — 稳定、务实、结果、物质

土是承载。它慢，但它是唯一能托住东西的元素。

- 核心领域：金钱、健康、工作成果、物质安全、身体、家庭根基、长期承诺
- 正向表达：踏实、耐心、可靠、有结果、善于积累、扎根、落地
- 阴影表达：固执、物质主义、停滞、抗拒变化、保守、沉重、困在安全区
- 在塔罗解读中：土元素牌多 → 这件事有结果，或者用户极度关注结果。土不在乎过程好不好看，土在乎"最后怎样"。问感情却土元素多 → 关系偏务实层面（经济、责任、现实匹配），少了一些火花和浪漫。土元素完全缺席 → 缺乏落地，想法和情绪很多但缺少"那个能接住的东西"——建议、计划、承诺、行动方案。
- 逆位：土的稳定性在瓦解——财务松动、安全感动摇、身体疲惫。但逆位土不全是坏：有时候需要松动才能重建（星币四逆→ 从死守到放手，可能是必要的解放）。
- 花色：星币 | 星座：金牛、处女、摩羯 | 守护星：金星
- 季节感：冬（沉淀与积累）

## 元素配对

两张牌放在一起，元素的互动比单看每张更有信息量。以下是常见配对模式，不是穷举——用它来启发解读，不机械套。

### 同元素

能量强化，该主题被反复敲打。两张火 → 行动力爆表，但也可能冲突升级。两张水 → 情感深但可能淹没。两张风 → 思维密集但可能停在纸上。两张土 → 稳但可能僵。

同元素牌的细微差异值得看：比如圣杯二（连接）与圣杯八（离开），同为水，一进一退——这组配对就在说情感层面有拉扯。

### 相邻元素

按占星相生逻辑排列：火→土→风→水→火。

- 火+土：行动产生结果。能量被接住、被落地。好配。
- 土+风：务实思考。规划与执行配合。好配。
- 风+水：思维引发情感波动，或情感需要表达。中性。
- 水+火：情感驱动行动。热情与直觉配合，但水可能浇灭火（过度谨慎），火可能蒸发水（冲动忽略感受）。

### 对立元素

火+水、风+土是天生张力对。

- 火+水：蒸汽——情绪化的行动（水被火烧开）或激情被情绪冷却（火被水浇灭）。视正逆位定方向。问感情时火+水常见：激情与情感的混合，可能是好事也可能是互相消耗。
- 风+土：沙尘——理性分析撞上现实壁垒，或务实态度限制思维发散。常见于事业牌阵：你有计划（风）但落地条件不成熟（土逆）。

### 缺对

一个元素完全没有，它的对立元素却在——这是失衡信号。

- 有火无水：行动力强但缺乏共情，可能伤人不自知。
- 有水无火：情感充沛但缺乏行动出口，容易陷在情绪里。
- 有风无土：想了很多但没有结果。
- 有土无风：埋头做但没想清楚为什么做。

## 元素缺失

某元素在牌阵中完全不出现，该类能量不足，需主动补足：

- 火缺失 → 行动力不足，需外部推动或主动点燃热情。
- 水缺失 → 情感流动受阻，需主动注入温暖与共情。问感情却水缺失尤其要点出——这段关系的情感水位在下降。
- 风缺失 → 思维不清或沟通不畅，需澄清想法、主动表达。可能用户一直在凭直觉或惯性走，没有停下来想清楚。
- 土缺失 → 缺乏落地与现实感，需关注务实层面。想法和情绪飘着，缺一个"能接住的人"或"能执行的计划"。

缺失不只是一种，有时两种元素同时缺席。比如火和土都缺 → 既没有行动力也没有结果落地——用户可能处于观望或无力状态。火和水都缺 → 既无激情也无情感连接——整件事可能是机械的或被迫的。

## 元素偏多

某元素在牌阵中重复出现，暗示能量失衡或该主题主导。但"多"要有阈值感：3 张同元素在小牌阵（3-5 张）中已经是强烈信号；在大牌阵（如凯尔特十字 10 张、黄道 12 张）中需要更多张才算。

- 火居多（≥3 张） → 行动与冲突密集，注意冲动和 burnout。好的一面：事情在推进，不会卡住。
- 水居多 → 情感流动主导，可能感性压过理性。好的一面：有温度、有连接、有直觉指引。
- 风居多 → 思维活跃但可能浮于空想，缺乏落地。风过多时建议回头看有没有土元素——没有的话，这些想法可能飞不走。
- 土居多 → 稳定但可能停滞，事件缺乏动力。尤其是三张及以上土元素同时出现——事情可能很稳，但也可能不动了。

## 元素与正逆位

逆位改变元素的表现方式，不是取消元素。一个逆位火元素牌仍然是火——只是烧的方式变了。

- 逆位火 → 行动停滞，热情受阻或方向不明。内在的火还在烧，但找不到出口，可能转为烦躁或自我消耗。
- 逆位水 → 情感流动停止，或情绪压抑/正在澄清。注意区分：压抑（水冻住了）和澄清（浑水在沉淀）是不同的阶段，前者需要融化，后者需要等待。
- 逆位风 → 思维混乱，或正从混乱转向清晰。风向变了——关键在于往哪个方向转。
- 逆位土 → 稳定感瓦解，正在重建安全感。旧地基松动，新地基还没打。

## 元素与方位综合

把元素状态和牌在牌阵中的位置（过去/现在/未来，或牌位含义）结合：

- 元素在"过去"位 → 这股能量从哪来、为当下埋了什么基调。
- 元素在"现在"位 → 当下被哪种能量主导、用户的注意力在哪。
- 元素在"未来"位 → 趋势往哪个方向走、需要准备什么。

例：逆位火出现在"过去"→ 过去的行动受阻，为当下埋下乏力基调；出现在"未来"→ 未来行动力可能不足，需提前准备。火出现在"现在"但水在"未来"→ 现在是冲刺期，未来更多是情感的收成或消化，别指望一直冲。

## 把元素状态挂钩用户问题

元素组合不能只描述"水缺失"，要落到对用户问题的具体含义。元素是诊断工具，不是装饰。

- 问事业却火缺失 → 点出需主动出击，不是时候等。
- 问感情却水缺失 → 点出情感流动需主动经营，关系的情感水位在降。
- 问决策却风缺失 → 点出还没想清楚，别急着做。
- 问感情却土缺失 → 点出关系缺乏现实落地，只有浪漫没有根基。
- 问事业却水过多 → 点出问题可能在人际关系或情绪上，不是在能力上。
- 问事业却风过多 → 点出可能想太多做太少，分析瘫痪。

同一个元素状态，换一个问题就换一个含义——必须回到用户问的是什么，元素才有指向。

## 小阿卡纳的发展逻辑

小阿卡纳(权杖/圣杯/宝剑/星币)的 14 张牌各有内在发展脉络,解读数字牌和宫廷牌时套这个框架:

### 数字牌 1-10：事物从开始到结束的完整发展过程

- 1(Ace) = 起点/种子,新能量的纯粹火花
- 2-3 = 萌芽,最初的展开与碰撞
- 4-6 = 中段,稳定、冲突、调整
- 7-9 = 深化或卡顿,接近转折
- 10 = 圆满,或过度后的循环终点(往往是"走到尽头需释放")

数字越大,该花色的能量越成熟也越沉重——1 是礼物,10 是承担。

### 宫廷牌：侍从 → 骑士 → 皇后 → 国王

某项能力或人生阶段的发展,从稚嫩探索到成熟掌控:

- 侍从(Page)= 初学者,好奇、试探、未定型
- 骑士(Knight)= 行动者,带着这股能量冲出去,但也可能冲动
- 皇后(Queen)= 内化的成熟,这股能量成为内在特质
- 国王(King)= 外显的掌控,这股能量被结构化地运用与表达

宫廷牌常代表一个人、一种人格侧面、或该花色能量在用户身上的成熟度。解读时问:这是谁?是用户自己,还是关系中的人?这个阶段走到了哪一步?
