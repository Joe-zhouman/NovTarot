# 元素系统与组合解读

解读涉及多张牌的元素关系时查这份。单张牌的元素属性在 `card-index.md` 和各单牌 ref 里。

## 四元素基本含义

- 火:行动力、热情、冲突、意志
- 水:情感、直觉、关系、潜意识
- 风:理性、思维、变革、沟通
- 土:稳定、务实、结果、物质

## 元素与花色 / 星座的对应

- 权杖(Wand):火 | 白羊座、狮子座、射手座 | 火星
- 圣杯(Cup):水 | 双鱼座、巨蟹座、天蝎座 | 月亮
- 宝剑(Sword):风 | 双子座、天秤座、水瓶座 | 水星
- 星币(Pentacle):土 | 金牛座、处女座、摩羯座 | 金星

## 组合解读要点

### 元素缺失

某元素在牌阵中完全不出现,预示该类能量不足,需主动补足:

- 火缺失 → 行动力不足,需外部推动或主动点燃热情
- 水缺失 → 情感流动受阻,需主动注入温暖与共情(问感情却水缺失尤其要点出)
- 风缺失 → 思维不清或沟通不畅,需澄清想法、主动表达
- 土缺失 → 缺乏落地与现实感,需关注务实层面

### 元素偏多

某元素在牌阵中重复出现,暗示能量失衡或该主题主导:

- 三张均为土 → 事件可能缺乏动力或陷入僵局,稳定但停滞
- 水居多 → 情感的流动主导,可能感性压过理性
- 火居多 → 行动与冲突密集,注意冲动
- 风居多 → 思维活跃但可能浮于空想,缺乏落地

### 元素与正逆位

逆位改变元素的表现方式,不是取消元素:

- 逆位火 → 行动停滞,热情受阻或方向不明
- 逆位水 → 情感流动停止,或情绪压抑/正在澄清
- 逆位风 → 思维混乱,或正从混乱转向清晰
- 逆位土 → 稳定感瓦解,正在重建安全感

### 元素与方位综合

把元素状态和牌在牌阵中的位置(过去/现在/未来,或牌位含义)结合。例:逆位火出现在"过去" → 过去的行动受阻,为当下埋下乏力基调;出现在"未来" → 未来行动力可能不足,需提前准备。

### 把元素状态挂钩用户问题

元素组合不能只描述"水缺失",要落到对用户问题的具体含义。问事业却火缺失 → 点出需主动出击;问感情却土缺失 → 点出关系缺乏现实落地。元素是诊断工具,不是装饰。
