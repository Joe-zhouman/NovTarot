# 行星含义

解读涉及牌的守护星时查这份。每张牌的守护星在 `card-index.md` 和各单牌 ref 里标注。

## 十大行星(含日月)

- **太阳**:内在自我与人格、生命力、显化
- **月亮**:本能与自我需求、情绪、潜意识、直觉
- **水星**:沟通和表达、思维、信息流动
- **金星**:个人魅力、价值、和谐、爱与美
- **火星**:行动与欲望、冲突、驱动力
- **木星**:个人成长、扩张、机遇、信念
- **土星**:局限性、结构、责任、考验、时间
- **天王星**:创造力、突变、觉醒、打破常规
- **海王星**:包容性、幻象、灵感、梦境、消融边界
- **冥王星**:黑暗面与潜力、转化、死亡与重生、深层力量

## 占星接入原则

守护星是牌的"能量来源",但接入解读时遵循:

1. **服务于问题,不堆砌**:不要罗列"这张牌对应金星"。要落到对用户问题的具体含义。
2. **守护星重叠时点出主题**:多张牌共享守护星(如星币五 + 教皇都由金星守护)→ 某主题被反复强调,指出来并落到具体含义(金星双重守护 → "价值重估")。
3. **行星与星座协同**:守护星是"什么能量",星座是"能量如何表达"。教皇 = 金星(价值)在金牛座(稳固、感官) → 对传统价值的稳固守护。
4. **无关联时不强行接入**:不是每张牌都需要占星解读。只在它真的照亮用户处境时用。

## 行星与心理动力的对应(荣格视角)

- 太阳/月亮 → 意识与潜意识的核心(人格面具 vs 阴影)
- 水星/金星 → 沟通与关系的功能
- 火星/土星 → 行动与限制的张力(本我冲动 vs 超我约束)
- 木星/土星 → 成长与考验的平衡
- 天王星/海王星/冥王星 → 超个人力量的三重奏:觉醒、消融、转化
